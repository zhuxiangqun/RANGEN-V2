在系统健康指标中记录实例池统计信息（已实现），需要确保在运行时正确记录。

### 2. 相关性计算并行化

修改 `_batch_calculate_relevance` 方法，即使证据数量 <= 3，也使用并行处理：

```python
# 当前代码（串行处理）
if len(evidence_list) <= 3:
    for ev in evidence_list:
        ev.relevance_score = self._calculate_relevance(ev, query)
    return evidence_list

# 优化后（并行处理）
async def calculate_relevance_async(ev):
    ev.relevance_score = self._calculate_relevance(ev, query)
    return ev

if len(evidence_list) <= 3:
    tasks = [calculate_relevance_async(ev) for ev in evidence_list]
    await asyncio.gather(*tasks)
    return evidence_list
```
