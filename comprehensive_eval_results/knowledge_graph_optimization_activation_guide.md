## 📋 如何激活优化

### 方法1: 重启知识图谱构建程序（推荐）

1. **停止当前程序**
   ```bash
   # 找到进程ID
   ps aux | grep build_knowledge_graph
   # 停止进程（替换PID为实际进程ID）
   kill <PID>
   ```

2. **重新启动程序**
   ```bash
   ./build_knowledge_graph.sh --batch-size 100 --resume
   ```

### 方法2: 检查环境变量（如果设置了）

如果环境变量`JINA_MAX_RETRIES`被设置为0，会覆盖默认值：

```bash
# 检查环境变量
echo $JINA_MAX_RETRIES

# 如果设置为0，取消设置或设置为3
unset JINA_MAX_RETRIES
# 或者
export JINA_MAX_RETRIES=3
```
