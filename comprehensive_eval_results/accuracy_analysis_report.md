## 🎯 问题根源分析

### 问题1：缺少系统答案（样本8）⚠️

**原因**：
1. 智能答案提取服务（`extract_core_answer_intelligently`）返回了`None`
2. 备用方案（`final_answer`或第一行）也没有成功
3. 导致没有记录"系统答案:"日志

**检查代码逻辑**：
```python
core_answer = await _extract_core_answer_intelligently(query_text, answer)
if core_answer:
    log_info(f"系统答案: {clean_answer}")
else:
    # 备用方案
    final_answer = getattr(result, 'final_answer', None)
    if final_answer:
        log_info(f"系统答案: {final_answer[:200]}")
    else:
        # 最后备用：第一行
        first_line = answer.split('\n')[0].strip()
        if first_line and first_line != "Reasoning Process:":
            log_info(f"系统答案: {first_line[:200]}")
```

**可能原因**：
- 答案提取服务可能过滤掉了答案（错误消息过滤过于严格）
- 或者答案内容确实不符合提取条件
