## 发现的问题

### 问题1：日志可能被覆盖
从`ResearchLogger`的代码可以看到：
```python
file_handler = logging.FileHandler(log_file, mode='w', encoding='utf-8')
```
- 使用`mode='w'`会覆盖日志文件
- 如果系统重新初始化，之前的日志会被清空

### 问题2：日志级别可能过滤了诊断日志
- 需要确认日志级别设置
- 确认诊断日志是否被正确输出

### 问题3：日志可能输出到不同的位置
- 可能有多个日志处理器
- 诊断日志可能输出到控制台而不是文件
