## ✅ 解决方案

### 方案1：改进提取逻辑（推荐）

**问题**：当前提取逻辑过于依赖特定格式的元数据字段，而实际元数据字段名不同。

**改进**：
1. **适配实际元数据字段**：
   - 将`query`改为`prompt`
   - 将`answer`改为`expected_answer`
   - 或者同时支持两种字段名

2. **使用LLM进行提取**：
   ```python

  "entities": [
    {{"name": "实体名", "type": "Person|Location|Organization|Event"}},
    ...
  ],
  "relations": [
    {{"entity1": "实体1", "entity2": "实体2", "relation": "关系类型"}},
    ...
  ]
}}"""
       # 调用LLM
       # 解析响应
       # 返回结构化数据
   ```

**如果知识条目来自FRAMES数据集**：
