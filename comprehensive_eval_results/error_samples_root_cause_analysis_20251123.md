## 🔍 错误分类

根据错误类型，将40个错误样本分为以下几类：

1. **数值错误** (8个): 数值不匹配
2. **答案不完整** (18个): 答案被截断或提取不完整

## 📊 详细分析

### 关键发现

通过分析40个错误样本，发现以下关键问题：

1. **答案提取问题**是最严重的问题（62.5%的错误）
   - 只提取核心实体，忽略完整描述
   - 只提取数字，忽略单位
   - 答案被长度限制截断

2. **知识检索问题**是第二严重的问题（12.5%的错误）
   - 检索不到相关证据（导致"unable to determine"）
   - 检索到错误证据（导致答案错误）

3. **数值提取问题**是第三严重的问题（20%的错误）
   - 提取了错误的数字
   - 只提取数字，忽略单位

4. **验证问题**导致部分样本返回"unable to determine"（7.5%的错误）

5. **格式差异**可能是可接受的（22.5%的错误）

#### 样本3: "100" vs "87"
**期望答案**: "87"  
**实际答案**: "100"  
**错误类型**: 数值错误

**根本原因分析**:
1. **知识检索问题**: 可能检索到了包含"100"的证据，但没有检索到包含"87"的证据
2. **答案提取问题**: 从多个数字中选择了错误的数字（可能选择了第一个或最大的数字）
3. **推理问题**: LLM可能基于错误的证据进行了推理

**代码位置**: `_extract_answer_simple()` (line 6107-6129)
```python

    numbers_clean = [n.replace(',', '') for n in numbers]
    numbers_float = [float(n) for n in numbers_clean]
```

**问题**: 如果响应中有多个数字，选择"最后出现的数字"可能不是正确答案
