## 🔍 根本原因分析

### 最可能的原因

**快速模型未初始化**，导致：

1. `fast_llm_integration`为`None`
2. 在`_select_llm_for_task`中，检查到`fast_llm`为`None`，直接返回推理模型
3. LLM复杂度判断代码未执行（因为`fast_llm_integration`不可用）
4. 所有查询都使用推理模型

**代码路径**:
```python
# 在 _select_llm_for_task 中
fast_llm = getattr(self, 'fast_llm_integration', None)
if not fast_llm:
    self.logger.warning("⚠️ [模型选择] 快速模型不可用，使用推理模型")
    return self.llm_integration  # 直接返回推理模型
```
