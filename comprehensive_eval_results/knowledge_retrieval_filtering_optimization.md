# 知识检索过滤优化

**优化时间**: 2025-11-09  
**问题**: 所有检索结果都被过滤（similarity_too_low: 15），导致无法获取有效知识

---

## 🔍 问题分析

### 日志显示
```
所有结果都被过滤，使用宽松模式重试验证（原始结果数: 15）
过滤原因统计: {'similarity_too_low': 15, 'entity_match_too_low': 2, 'keyword_match_too_low': 1, 'is_question': 0, 'content_too_short': 0}
```

### 根本原因

1. **双重过滤问题**：
   - 知识库管理系统层面：`query_knowledge`使用`similarity_threshold=0.40`过滤
   - 应用层面：`_validate_result_multi_dimension`再次使用`similarity_threshold=0.40`验证
   - 导致所有相似度低于0.40的结果都被过滤

2. **阈值过高**：
   - 基础阈值0.40可能对于某些查询来说太高
   - 即使宽松模式降低到0.20，如果所有结果的相似度都低于0.20，还是会被过滤

3. **设计理念问题**：
   - 应该在检索阶段获取更多候选结果
   - 在应用层进行更智能的验证，而不是在检索阶段就过度过滤

---

## 🔧 优化方案

### 1. 降低基础阈值（已完成）

**改进前**：
```python
base_threshold = 0.40  # 统一阈值
```

**改进后**：
```python
base_threshold = 0.30  # 统一阈值（从0.40降低到0.30）
```

**理由**：
- 让更多结果通过，由LLM判断相关性
- 而不是在检索阶段过度过滤

### 2. 降低宽松模式阈值（已完成）

**改进前**：
```python
relaxed_threshold = max(0.20, optimized_threshold - 0.20)  # 降低0.20作为宽松阈值
```

**改进后**：
```python
relaxed_threshold = max(0.15, optimized_threshold - 0.25)  # 降低0.25作为宽松阈值，最低0.15
```

**理由**：
- 在宽松模式下，应该尽可能让结果通过
- 由LLM判断相关性，而不是在验证阶段过度过滤

### 3. 分离检索阈值和验证阈值（新增）

**改进前**：
```python
results = self.kms_service.query_knowledge(
    query=query,
    similarity_threshold=optimized_threshold,  # 0.30
    ...
)
```

**改进后**：
```python
# 检索阶段使用更低的阈值（0.20），获取更多候选结果
retrieval_threshold = 0.20
results = self.kms_service.query_knowledge(
    query=query,
    similarity_threshold=retrieval_threshold,  # 0.20（更宽松）
    ...
)
# 然后在应用层使用optimized_threshold（0.30）进行验证
```

**理由**：
- **检索阶段**：使用更低的阈值（0.20），获取更多候选结果
- **验证阶段**：使用较高的阈值（0.30），进行更严格的验证
- 这样可以避免在检索阶段就过滤掉所有结果

## 📊 优化效果

### 改进前
- 检索阈值：0.40
- 验证阈值：0.40
- 宽松模式阈值：0.20
- **结果**：所有15个结果都被过滤

### 改进后
- 检索阈值：0.20（更宽松，获取更多候选）
- 验证阈值：0.30（适中，进行验证）
- 宽松模式阈值：0.15（更宽松，让更多结果通过）
- **预期结果**：更多结果能够通过验证

---

## 🎯 设计理念

### 核心原则

1. **检索阶段应该宽松**：
   - 使用较低的阈值（0.20），获取更多候选结果
   - 避免在检索阶段就过滤掉所有结果

2. **验证阶段应该智能**：
   - 使用适中的阈值（0.30），进行多维度验证
   - 由LLM判断相关性，而不是过度依赖相似度

3. **宽松模式应该更宽松**：
   - 使用更低的阈值（0.15），让更多结果通过
   - 由LLM判断相关性，而不是在验证阶段过度过滤
