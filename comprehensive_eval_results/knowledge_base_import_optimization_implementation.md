## 🎯 性能提升计算

### Wikipedia抓取时间

**优化前**:
- 每个item平均3个链接
- 每个链接: 0.5秒延迟 + 2-3秒请求 = 2.5-3.5秒
- 串行处理: 3 × 3秒 = 9秒
- 加上重试: **15-20秒/item**

**优化后**:
- 每个item平均3个链接
- 每个链接: 0.2秒延迟 + 2-3秒请求 = 2.2-3.2秒
- 并发处理（5个并发）: max(3秒) = **3-4秒/item**
- 加上重试: **6-10秒/item**

**提升**: 减少60-70%的抓取时间
