### 改进1: 智能截断策略

```python
def _smart_truncate_for_embedding(self, text: str, max_length: int) -> str:
    """智能截断文本用于向量化"""
    
    # 策略1: 优先保留排名列表
    if "list" in text.lower() or "ranking" in text.lower():
        ranking_section = self._extract_ranking_section(text)
        if ranking_section:
            return ranking_section  # 返回完整的排名列表
    
    # 策略2: 保留开头和结尾
    first_part = text[:max_length // 2]
    last_part = text[-max_length // 2:]
    return first_part + "\n\n[...]\n\n" + last_part
```

**效果**:
- ✅ 对于排名列表类内容，优先提取排名列表部分
- ✅ 确保包含"37th"、"823 feet"等关键信息
- ✅ 比简单截断更智能
