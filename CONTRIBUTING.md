# 🤝 贡献指南

感谢您对RANGEN项目的关注和支持！我们欢迎各种形式的贡献，包括代码、文档、测试、功能建议和问题反馈。

## 📋 贡献流程

### 1. 报告问题
如果您发现bug或有功能建议，请：
1. 在GitHub Issues中搜索是否已有相关讨论
2. 如果没有，创建一个新的Issue
3. 提供清晰的问题描述和复现步骤
4. 如果是功能建议，说明使用场景和预期效果

### 2. 开发环境搭建
请参考 [开发环境搭建文档](docs/development/development-environment.md) 配置本地开发环境。

### 3. 代码规范
请遵循项目的 [代码规范指南](docs/development/code-style-guide.md)，确保代码质量一致。

### 4. 提交代码
1. Fork项目仓库
2. 创建功能分支（命名规范：`feature/功能描述` 或 `fix/问题描述`）
3. 编写代码并添加测试
4. 确保所有测试通过
5. 提交代码，遵循提交消息规范（见下文）

### 5. 创建Pull Request
1. 确保代码符合项目规范
2. 提供清晰的PR描述
3. 关联相关的Issue（如适用）
4. 等待代码审查

## 🧪 测试要求
所有贡献的代码必须包含相应的测试：
- **单元测试**：测试核心功能逻辑
- **集成测试**：测试组件间协作
- **性能测试**：确保性能符合要求
- **安全测试**：验证安全功能正确性

运行测试命令：
```bash
# 运行所有测试
pytest

# 运行特定测试
pytest tests/test_specific_file.py
```

## 📝 提交消息规范
请使用以下格式的提交消息：
```
类型(范围): 简要描述

详细描述（可选）

关联Issue: #123
```

### 提交类型
- `feat`: 新功能
- `fix`: 修复bug
- `docs`: 文档更新
- `style`: 代码格式调整
- `refactor`: 重构代码
- `test`: 测试相关
- `chore`: 构建或辅助工具更新

### 示例
```
feat(auth): 添加JWT令牌认证支持

- 实现JWT令牌生成和验证
- 添加令牌刷新机制
- 更新API认证文档

关联Issue: #45
```

## 📚 文档贡献
我们重视文档质量，欢迎贡献：
1. **技术文档**：API文档、架构说明、配置指南
2. **用户文档**：使用指南、教程、最佳实践
3. **开发文档**：开发环境、代码规范、测试指南

文档要求：
- 使用中文撰写，保持语言清晰简洁
- 优先说明原理和设计思路，代码示例适量
- 遵循现有文档结构和风格
- 确保所有链接有效

## 🔍 代码审查标准
所有提交的代码都会经过审查，审查关注点：
1. **功能正确性**：代码按预期工作
2. **代码质量**：遵循代码规范，可读性好
3. **测试覆盖**：包含必要的测试用例
4. **文档完整性**：更新相关文档
5. **性能影响**：不降低系统性能
6. **安全性**：不引入安全漏洞

## 🛠️ 开发工具推荐
- **代码编辑器**：VS Code、PyCharm
- **Python环境**：Python 3.9+
- **依赖管理**：pip、poetry
- **测试框架**：pytest
- **代码格式化**：black、isort
- **代码检查**：pylint、mypy

## 📊 贡献者权益
所有贡献者都将：
1. 在贡献者名单中被提及
2. 获得项目的优先使用支持
3. 参与项目发展方向讨论
4. 根据贡献程度获得相应的认可

## ❓ 常见问题

### Q: 如何开始第一个贡献？
A: 建议从以下类型开始：
1. 修复文档中的错别字或错误链接
2. 解决标记为"good first issue"的Issue
3. 添加缺少的测试用例
4. 改进代码注释和文档

### Q: 贡献需要签署CLA吗？
A: 目前不需要，但所有贡献者需要同意将贡献内容授权给项目使用。

### Q: 如何获得开发帮助？
A: 可以通过以下方式：
1. 查看现有文档和代码示例
2. 在GitHub Discussions中提问
3. 查看Issue中的相关讨论
4. 联系核心维护团队

### Q: 我的贡献被拒绝了怎么办？
A: 请理解代码审查是为了保证项目质量。如果被拒绝：
1. 仔细阅读拒绝原因和建议
2. 根据反馈修改代码
3. 重新提交或进一步讨论
4. 如有疑问，与审查者友好沟通

- **GitHub Discussions**: 技术讨论和开发帮助
- **项目维护者**: 直接联系核心团队成员
