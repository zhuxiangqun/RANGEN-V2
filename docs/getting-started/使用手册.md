## 第1章：系统准备

### 1.1 系统要求

#### 硬件要求
| 项目 | 最低要求 | 推荐配置 | 说明 |
|------|----------|----------|------|
| CPU | 4核 | 8核+ | 支持多线程处理 |
| 内存 | 8GB | 16GB+ | 用于运行多个智能体 |
| 存储 | 20GB | 50GB+ | 包含模型缓存和日志 |
| 网络 | 100Mbps | 1Gbps+ | 外部API调用需要 |

#### 软件要求
- **操作系统**: Ubuntu 20.04+, CentOS 8+, macOS 12+, Windows 10+ (WSL2)
- **Python版本**: 3.9, 3.10, 3.11 (推荐3.10)
- **包管理工具**: pip 21.0+
- **容器环境**: Docker 20.10+, Docker Compose 2.0+ (可选)

#### 第三方服务
- **LLM提供商API密钥** (至少一个):
  - DeepSeek API密钥
  - Step-3.5-Flash API密钥 (通过OpenRouter/NVIDIA NIM)
  - OpenAI API密钥 (可选)
- **向量数据库** (可选):
  - ChromaDB (内置)
  - Weaviate, Pinecone等 (需要额外配置)

## 第3章：核心功能详解

### 3.1 智能体协作系统

#### 3.1.1 智能体类型和使用场景

**1. 首席智能体 (Chief Agent)**
- **功能**: 总体协调，任务分解，结果整合
- **使用场景**: 
  ```bash
  # 复杂问题分解
  curl -X POST http://localhost:8000/chat \
    -H "Content-Type: application/json" \
    -d '{
      "query": "请分析量子计算对密码学的影响，并给出具体应用场景",
      "agent_mode": "chief",
      "max_depth": 3
    }'
  ```

**2. 推理智能体 (Reasoning Agent)**
- **功能**: 逻辑推理，假设检验，因果分析
- **使用场景**:
  ```bash
  # 逻辑推理问题
  curl -X POST http://localhost:8000/chat \
    -H "Content-Type: application/json" \
    -d '{
      "query": "如果所有猫都怕狗，Tom是猫，那么Tom怕狗吗？请解释推理过程",
      "agent_mode": "reasoning"
    }'
  ```

**3. 检索增强智能体 (RAG Agent)**
- **功能**: 知识检索，事实核查，引用管理
- **使用场景**:
  ```bash
  # 需要事实核查的问题
  curl -X POST http://localhost:8000/chat \
    -H "Content-Type: application/json" \
    -d '{
      "query": "2023年诺贝尔物理学奖的获奖者是谁？他们的主要贡献是什么？",
      "agent_mode": "rag",
      "citation_required": true
    }'
  ```

#### 3.1.2 智能体组合使用

**多智能体协作示例**:
```python
from src.agents import AgentFactory

# 创建智能体组合
chief_agent = AgentFactory.create_agent("chief")
reasoning_agent = AgentFactory.create_agent("reasoning")
rag_agent = AgentFactory.create_agent("rag")

# 协作解决复杂问题
async def solve_complex_problem(query):
    # 首席智能体分解任务
    subtasks = await chief_agent.decompose_task(query)
    
    results = []
    for subtask in subtasks:
        if subtask["type"] == "reasoning":
            # 推理智能体处理逻辑问题
            result = await reasoning_agent.process(subtask["content"])
        elif subtask["type"] == "fact_check":
            # RAG智能体处理事实核查
            result = await rag_agent.process(subtask["content"])
        else:
            # 默认处理
            result = await chief_agent.process(subtask["content"])
        
        results.append(result)
    
    # 整合结果
    final_result = await chief_agent.integrate_results(results)
    return final_result
```

### 3.4 工作流系统

#### 3.4.1 LangGraph工作流

**标准工作流结构**:
```python
from langgraph.graph import StateGraph
from src.core.langgraph_unified_workflow import UnifiedResearchWorkflow

# 创建工作流
workflow = StateGraph(ResearchState)

# 定义节点
workflow.add_node("query_analysis", analyze_query)
workflow.add_node("knowledge_retrieval", retrieve_knowledge)
workflow.add_node("reasoning", perform_reasoning)
workflow.add_node("validation", validate_results)
workflow.add_node("integration", integrate_results)

# 定义边（执行路径）
workflow.set_entry_point("query_analysis")
workflow.add_edge("query_analysis", "knowledge_retrieval")
workflow.add_conditional_edges(
    "knowledge_retrieval",
    decide_next_step,
    {
        "needs_reasoning": "reasoning",
        "direct_answer": "integration"
    }
)
workflow.add_edge("reasoning", "validation")
workflow.add_edge("validation", "integration")

# 编译工作流
compiled_workflow = workflow.compile()
```

#### 3.4.2 自定义工作流

**创建自定义工作流**:
```python
# custom_workflow.py
from src.core.langgraph_unified_workflow import BaseWorkflow

class CustomResearchWorkflow(BaseWorkflow):
    def __init__(self, config=None):
        super().__init__(config)
        self.custom_nodes = {}
    
    def add_custom_node(self, name, function):
        """添加自定义节点"""
        self.custom_nodes[name] = function
        self.graph.add_node(name, function)
    
    def set_custom_flow(self, flow_definition):
        """设置自定义执行流程"""
        # flow_definition示例: ["node1", "node2", {"node3": "conditional"}]
        self.apply_flow_definition(flow_definition)
```

**使用自定义工作流**:
```python
from custom_workflow import CustomResearchWorkflow

# 创建实例

    "max_iterations": 5,

    "domain_expert",

)
```

### 4.3 系统集成

#### 4.3.1 API集成

**Python客户端**:
```python
# rangen_client.py
import requests

class RANGENClient:
    def __init__(self, base_url="http://localhost:8000", api_key=None):
        self.base_url = base_url
        self.api_key = api_key
        self.session = requests.Session()
        
        if api_key:
            self.session.headers.update({
                "Authorization": f"Bearer {api_key}"
            })
    
    def query(self, question, **kwargs):
        """执行查询"""
        response = self.session.post(
            f"{self.base_url}/chat",
            json={"query": question, **kwargs}
        )
        response.raise_for_status()
        return response.json()
    
    def batch_query(self, questions, **kwargs):
        """批量查询"""
        response = self.session.post(
            f"{self.base_url}/chat/batch",
            json={"queries": questions, **kwargs}
        )
        response.raise_for_status()
        return response.json()
```

**使用示例**:
```python
client = RANGENClient(api_key="your_api_key")

# 单次查询
result = client.query("什么是量子计算？")
print(f"答案: {result['answer']}")

# 批量查询
results = client.batch_query([
    "问题1",
    "问题2",
    "问题3"
])
```
