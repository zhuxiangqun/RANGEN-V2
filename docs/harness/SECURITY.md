# RANGEN 安全约束

> 详细说明安全策略和防护措施

---

## 安全原则

### 1. 最小权限

- Agent 只访问完成任务所需的最小资源
- 禁止访问生产数据库
- 禁止执行危险命令

### 2. 沙箱隔离

- Agent 在隔离环境中运行
- 每次任务独立环境
- 网络访问受限

### 3. 审计追踪

- 所有操作记录日志
- 可追溯到执行者
- 异常行为告警

---

## 工具策略

### 风险等级

| 等级 | 类型 | 需要审批 |
|------|------|---------|
| 低 | 读取文件、搜索 | ❌ |
| 中 | 写文件、执行测试 | ⚠️ |
| 高 | 删除数据、执行命令 | ✅ |

### Tool Policy

- 位置: `src/agents/tools/tool_policy.py`
- 功能: 风险评估、策略引擎控制

---

## 禁止模式

```
❌ 禁止:
- rm -rf /
- 删除生产数据
- 提交密钥到 Git
- 执行未知来源代码
```

---

## 响应机制

```
检测到危险操作 → 立即终止 → 告警通知 → 人工复核
```
