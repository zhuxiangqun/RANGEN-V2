## 优化建议

### 1. 如果测试时间过长

可以修改测试脚本，跳过实际执行：

```python
# 在 tests/run_optimization_tests.py 中添加
SKIP_EXECUTION = os.getenv('SKIP_EXECUTION', 'false').lower() == 'true'

async def test_1_persistent_checkpoint(self):
    if SKIP_EXECUTION:
        logger.info("⏭️ 跳过实际执行（仅测试初始化）")
        return
    # ... 原有代码
```

然后运行：
```bash
SKIP_EXECUTION=true python tests/run_optimization_tests.py
```

### 2. 使用 Mock LLM

创建 Mock LLM 响应，避免实际 API 调用（需要修改代码）。

### 3. 减少测试查询复杂度

使用更简单的查询，减少 API 调用次数。

### 4. 单独运行测试

只运行特定的测试，而不是全部：

```python
# 修改 main() 函数
async def main():
    runner = TestRunner()
    await runner.setup()
    try:
        # 只运行测试1
        await runner.run_test("测试1：持久化检查点", runner.test_1_persistent_checkpoint)
    finally:
        runner.cleanup()
```
