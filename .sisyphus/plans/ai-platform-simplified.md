# RANGEN AI中台 - 简化版方案

## TL;DR

> **目标**：将现有RANGEN系统作为AI中台，供其他应用通过自然语言调用AI能力

> **核心功能**：
> - API接口暴露（现有 `/chat` 增强）
> - API Key 认证
> - Python/JS SDK
> - 可选：简单管理后台

> **预估工作量**：1-2周

---

## 三、API设计

### 核心接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/execute` | POST | 自然语言执行任务 |
| `/api/v1/chat` | POST | 简单对话 |
| `/api/v1/agents` | GET | 列出可用智能体 |
| `/api/v1/agents/{id}/execute` | POST | 执行特定智能体 |
| `/api/v1/knowledge/query` | POST | 知识库检索 |
| `/api/v1/knowledge/upload` | POST | 上传文档到知识库 |

### 认证

```
Header: Authorization: Bearer {api_key}
```

## 五、使用示例

### Python SDK

```python
from rangenai import RangenClient

# 初始化客户端
client = RangenClient(
    api_key="rangen_sk_xxxxx",
    base_url="http://localhost:8000"
)

# 自然语言任务执行
result = client.execute("""
    帮我测试 ./src/calc.py 这个Python程序：
    1. 检查是否有bug
    2. 运行一下看看结果
    3. 给出测试报告
""")

print(result)
# AI中台自动完成：
# - 读取 calc.py 代码
# - 分析潜在bug
# - 执行测试用例
# - 生成测试报告
```

### 性能测试

```python
result = client.execute("""
    对 ./src/fibonacci.py 做性能测试：
    1. 测试 n=30 的执行时间
    2. 分析内存使用
    3. 给出优化建议
""")
```

### 知识库问答

```python
# 上传文档
client.knowledge.upload(
    file_path="./docs/api_spec.pdf"
)

# 基于知识库问答
result = client.execute(
    "你们公司的API是怎么认证的？",
    knowledge=True  # 启用知识库检索
)
```
