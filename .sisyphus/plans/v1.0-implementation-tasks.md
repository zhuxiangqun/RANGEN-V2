## 任务清单

### 阶段1：数据库和存储层 (第1天)

- [ ] 1.1 创建SQLite数据库表

  **SQL文件**: `migrations/v1.0_create_tables.sql`
  ```sql
  CREATE TABLE IF NOT EXISTS agents (
      id TEXT PRIMARY KEY,
      name TEXT NOT NULL,
      type TEXT NOT NULL DEFAULT 'agent',
      description TEXT,
      config_path TEXT,
      status TEXT DEFAULT 'active',
      created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
      updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
      reference_count INTEGER DEFAULT 0
  );

  CREATE TABLE IF NOT EXISTS skills (
      id TEXT PRIMARY KEY,
      name TEXT NOT NULL,
      description TEXT,
      tools TEXT,
      config_path TEXT,
      status TEXT DEFAULT 'active',
      created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
      reference_count INTEGER DEFAULT 0
  );

  CREATE TABLE IF NOT EXISTS tools (
      id TEXT PRIMARY KEY,
      name TEXT NOT NULL,
      description TEXT,
      type TEXT,
      status TEXT DEFAULT 'active',
      created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
  );

  CREATE INDEX idx_agents_status ON agents(status);
  CREATE INDEX idx_skills_status ON skills(status);
  ```

- [ ] 1.2 创建数据库连接模块

  **文件**: `src/services/database.py`
  - SQLite连接管理
  - 初始化数据库
  - 提供CRUD操作接口

- [ ] 1.3 初始化现有Tools到数据库

  **任务**: 扫描现有Tool实现，导入到tools表

### 阶段2：Agent CRUD API (第2天)

- [ ] 2.1 创建数据模型

  **文件**: `src/api/models_agent.py`
  ```python
  class AgentCreate(BaseModel):
      name: str
      description: Optional[str] = None
      type: str = "agent"
      skills: List[str] = []
      tools: List[str] = []
      config: Dict = {}

  class AgentResponse(BaseModel):
      id: str
      name: str
      type: str
      description: Optional[str]
      skills: List[str]
      tools: List[str]

      created_at: datetime
      updated_at: datetime
  ```

### 阶段5：能力检查器 (第3-4天)

  **文件**: `src/services/capability_checker.py`

  ```python
  class CapabilityChecker:
      """能力检查器 - 创建新Agent前检查现有能力"""
      
      def check_tools(self, requirements: List[str]) -> CheckResult:
          """检查现有Tools是否满足需求"""
          
      def check_skills(self, requirements: List[str]) -> CheckResult:
          """检查现有Skills是否满足需求"""

          """检查现有Agents是否满足需求"""
  ```
